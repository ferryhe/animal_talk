## Naming Convention / 命名规范

Files are named using the format: `YYYYMMDD-description.md`

文件使用格式命名：`YYYYMMDD-描述.md`

- `YYYYMMDD`: Date when the document was created or last significantly updated
- `description`: Brief description of the document content
- Language suffix (`-zh`, `-en`) when necessary
